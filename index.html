Batocera e Jogos Retro

Apresentação do Projeto

O Batocera é uma distribuição Linux voltada para emulação de jogos retro. É fácil de usar e suporta uma ampla gama de consoles, tornando-o uma escolha ideal para reviver clássicos dos videogames.

Sobre o Projeto

Este projeto integra o sistema operacional Batocera, uma distribuição Linux voltada para emulação de jogos retro. O objetivo é criar uma máquina de jogos que permita reviver os clássicos do passado de maneira fácil e acessível.

Clique nos nomes dos consoles abaixo para ver mais detalhes:

  • Atari
  • M.A.M.E
  • Nintendo
  • Game Boy

Apresentação do Console

É um pc/console criado para operar em Linux Batocera e rodar jogos retrô. Sua montagem em MDF reforçado foi criada com o propósito de remeter a um fliperama

Atari

Características

  • Processador: 1.19 MHz
  • Gráficos: 160 x 192 pixels
  • Som: 2 canais de áudio
  • Controles: Joystick

Jogos Principais

  • Pong
  • Space Invaders
  • Asteroids
  • Pac-Man
  • Adventure

Curiosidades

  • O Atari 2600 foi lançado em 1977.
  • Foi um dos primeiros consoles a usar cartuchos de jogos intercambiáveis.
  • O jogo Pac-Man para Atari 2600 foi um dos primeiros grandes fracassos comerciais da indústria de videogames.
  • O Atari 2600 é frequentemente creditado por popularizar os videogames em casa.
  • Space Invaders foi um dos primeiros grandes sucessos do Atari 2600 e ajudou a solidificar o console como um ícone da cultura pop dos anos 80.

M.A.M.E

Características

  • Processador: I3 8°
  • Memoria: 4gb Ram ddr4
  • Placa Grafica: Placa gráfica integrada suficiente para processamento de gráficos 2D e 3D básicos. Intel UHD Graphics 630, AMD Radeon Vega 8
  • Armazenamento: SSD ou HDD com pelo menos 120 GB de espaço livre.

Jogos Principais

  • Metal Slug
  • Donkey Kong
  • Mortal Kombat
  • Galaga
  • Street Fighter Alpha3

Curiosidades

  • Origem Histórica: M.A.M.E. começou como um projeto para preservar a história dos jogos de arcade, ajudando a manter vivas essas experiências de gameplay para as futuras gerações.
  • Comunidade Ativa: Tem uma comunidade robusta de entusiastas que contribuem para aprimorar o software, adicionar novos recursos e corrigir problemas.
  • Desafios Legais: O uso de ROMs e BIOSs de arcade pode ser uma área cinzenta legalmente, dependendo da jurisdição e da propriedade intelectual envolvida.

Nintendo Entertainment System (NES)

Características

  • Processador: 1.79 MHz
  • Gráficos: 256 x 240 pixels
  • Som: 5 canais de áudio
  • Controles: Gamepad

Jogos Principais

  • Super Mario Bros.
  • The Legend of Zelda
  • Metroid
  • Castlevania
  • Mega Man

Curiosidades

  • O NES foi lançado em 1983 no Japão como Famicom e em 1985 na América do Norte.
  • Super Mario Bros. se tornou um dos jogos mais vendidos de todos os tempos.
  • The Legend of Zelda introduziu muitos conceitos que se tornaram padrão em jogos de aventura.
  • O NES revitalizou a indústria de videogames após a crise de 1983 nos EUA, causada pelo mercado saturado de jogos de baixa qualidade.
  • O NES teve uma vasta biblioteca de jogos que incluía muitos clássicos que definiram gêneros inteiros de jogos.

Game Boy

Características

  • Processador: 4.19 MHz
  • Gráficos: 160 x 144 pixels
  • Som: 4 canais de áudio
  • Controles: Gamepad

Jogos Principais

  • Tetris
  • Super Mario Land
  • Pokémon Red/Blue
  • The Legend of Zelda: Link's Awakening
  • Metroid II: Return of Samus

Curiosidades

  • O Game Boy foi lançado pela Nintendo em 1989.
  • Tetris, um dos jogos mais populares do Game Boy, ajudou a consolidar o sucesso mundial do console.
  • O Game Boy foi o primeiro console portátil a oferecer uma biblioteca de jogos extensa e diversificada.
  • Pokémon Red/Blue foi um dos primeiros jogos a usar a funcionalidade de link do Game Boy para trocar e batalhar Pokémon entre jogadores.
  • O Game Boy teve um impacto duradouro na cultura pop e é considerado um dos consoles mais importantes da história dos videogames.
